## Introduction

MJ Chatbot is a powerful AI-driven chatbot application that leverages the capabilities of Flutter and the ChatGPT-3 model. It provides users with a seamless conversational experience, where they can interact with the chatbot as if they were talking to another person. Whether you need quick information, recommendations, translations, or simply want to engage in a casual conversation, MJ Chatbot has got you covered.

## Installation

### Prerequisites

Before installing MJ Chatbot, ensure that you have the following prerequisites:

- Flutter SDK (version X.X.X): Follow the official Flutter installation guide to set up Flutter on your machine.
- ChatGPT-3 API access: Obtain API credentials from OpenAI to use the ChatGPT-3 model.

### Steps

1. Clone the MJ Chatbot repository:

   ```bash
   git clone https://github.com/your-username/mj-chatbot.git
   ```

2. Navigate to the project directory:

   ```bash
   cd mj-chatbot
   ```

3. create missing files:

   ```bash
   flutter create .
   ```
   
4. Install the required dependencies:

   ```bash
   flutter pub get
   ```

5. Configure the ChatGPT-3 API credentials:

   - Open the `lib/constants.dart` file.
   - Replace `apiKey` with your actual ChatGPT-3 API key.
   - Replace `apikey` in getNews function with your actual NewsApi API key.

6. Build and run the MJ Chatbot app on your desired platform:

   ```bash
   flutter run
   ```

## Inspiration and Technical Challenges
The inspiration behind the MJ Chatbot app came from the desire to develop an AI-powered chatbot that could assist users in various tasks and provide valuable information. I wanted to create a chatbot that feels more like conversing with a human rather than a machine. This presented the challenge of integrating the GPT-3 model effectively and fine-tuning it to act in a specific way.

During the development process, I encountered technical challenges that pushed me to expand my skills and knowledge. One of the main challenges was learning how to use machine learning models effectively within a mobile application. As a developer with no prior experience in machine learning, I had to invest time in studying the documentation and understanding the intricacies of the GPT-3 API.

Another challenge was integrating the Flutter framework with the AI model and ensuring smooth communication between the user interface and the model's response generation. This required careful architecture design and efficient data handling to provide quick and accurate responses to user queries.

Additionally, I faced the challenge of enhancing the chatbot's capabilities beyond its default functionalities. I invested time in extending its abilities, such as adding some real-time data updates.
